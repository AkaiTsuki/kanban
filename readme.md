# React Kanban App

## Pre-request:
```
npm install -g webpack
```

## How to Run
### Start back end server
```
From project directory:
cd kanban-core
npm install
npm start
```

### Start webpack-dev-server
on windows 10 or MacOS
```
From project directory:
cd kanban-ui
npm install
npm start
```
Open browser http://localhost/8080

on windows 7
```
From project directory:
cd kanban-ui
npm install

set PORT=9000
npm start
```
Open browser(No IE support) http://localhost:9000
