# React Kanban App

## Pre-request:
```
npm install -g webpack
```

## How to Run
### Start back end server
```
From project directory:
cd kanban-core
npm install
npm start
```

### Start webpack-dev-server
```
From project directory:
cd kanban-ui
npm install
npm start
```
Do not use IE Browser, access http://localhost/8080
If cannot see home page:
```
set PORT=9000
npm start
```
Then http://localhost:9000
